Git and Github

Quick setup

Create and push a new repository on the command line

echo "# RepoName" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main (in case your default branch is not main)
git remote add origin https://github.com/MochibagelBaby/RepoName.git
git push -u origin main
